# Regras de Leitura de Placa
- Extrair texto bruto e versão normalizada separadamente.
- Sinalizar ruído OCR quando caracteres críticos forem ambíguos (0/O, 1/I, 5/S).
- Manter rastreabilidade: o que foi lido, o que foi inferido.
- Se houver mais de uma tensão/frequência, tratar como configuração alternativa até validação.
